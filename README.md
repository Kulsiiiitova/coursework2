# Проект по реализации получения информации о вакансиях с платформы hh.ru
## Описание

Проект работает с классами, у которых есть атрибуты, описывающие классы

1. Клонируйте репозиторий:
```
git clone https://github.com/username/coursework_2.git
```

## Использование:

1. Откройте проект в вашей среде разработки.
2. Запустите файл main.py
```
python main.py
```
3. Добавьте новые тестовые файлы и запустите с помощью команды:

```
python -m tests.name_file
```
## Модуль `Vacancy`:
Используется для создания абстрактных классов `BaseProduct` для класса `Product` и `BaseCategory` для классов `Category` и `Order`.

## Модуль `product`:

Используется для создания вакансий `Vacacny`.
Имеют атрибуты: `name, url, description, requirements, salary`. Имеет возможность сравнивать вакансии по максимальной зарплате.

## Модуль `job_vacancy_data`:

Используется для добавления, удаления и поиска вакансий в файле.
Имеет атрибут: `file_path`. 

## Модуль `classAPI`:
Используется для работы с API платформой hh.ru с вакансиями. Получает вакансии по запросу.
